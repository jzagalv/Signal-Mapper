# Signal Mapper (PyQt5)

## Ejecutar
```bash
pip install -r requirements.txt
python app.py
```

## Sin proyecto abierto
- Botón **Nuevo proyecto…**
- Botón **Abrir proyecto…**
- Botón **Biblioteca global de señales…** (se guarda en `template_library.json` junto a `app.py`)

## Con proyecto abierto
- **Proyecto → Nueva bahía…**
- **Proyecto → Nuevo equipo…**
- **Plantillas → Importar biblioteca global al proyecto**
- Dock de plantillas permite editar: **Fuente: Proyecto / Global**

## Replicación
- Proyecto → Replicar bahía… (copia equipos + layout + señales; enlaces fuera de la bahía quedan PENDIENTES)

- Proyecto → Replicar bahía… ahora permite reemplazo por token (ej. H1→H2) y marca externos como PENDIENTE.
